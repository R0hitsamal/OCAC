Server Setup
Step-1 => mkdir myapp
Step-2 => cd myapp
Step-3 => npm init
Step-4 => create a file  inside your project app.js
Step-5 => node app.js
Step-6 => set script inside package.json "start" : "node app.js"

DB Connection
1-npm i mongoose
2- require mongoose
3- mongoose.connect()
4- new mongoose.schema()
5- mongoose.model()
